### 环境说明

目前 DynamicTp 有不少特性是基于 Spring 做的，所以强依赖 Spring 环境。

### Java 版本

支持 Java 8 到 Java 17 版本

### Spring 版本

支持 Spring 4.x、5.x、6.x 版本

### SpringBoot 版本

核心功能支持 SpringBoot 1.x、2.x、3.x 版本，部分功能 1.x 支持的不是很好
